设计规范说明

配色方案

主色: #E8836B 珊瑚橘 - 温暖而不刺眼
辅色: #7EB8A2 薄荷绿 - 清新点缀
暖杏: #F5C5A3 - 渐变过渡
背景: #FFF8F3 奶油色 / #FAF3ED 暖灰
文字: #4A3228 深棕 / #6B5248 正文
辅助: #A89088 灰棕

圆角规范

大卡片: 24rpx (Banner、弹窗)
中卡片: 16rpx (商品卡、功能区)
小元素: 10rpx (图标、标签)
按钮: 48rpx (全圆角胶囊)
底部导航: 无圆角，保持规整

投影规范

轻投影: 0 2px 8px rgba(200,140,110,0.08)
标准投影: 0 4px 16px rgba(200,140,110,0.12)
强调投影: 0 8px 30px rgba(200,140,110,0.16)
使用暖色调阴影，避免冰冷的灰色投影

字体规范

大标题: 36rpx Bold
区块标题: 30rpx Bold
商品名: 28rpx Regular
辅助文字: 24rpx Regular
小标签: 20rpx Semibold
价格使用: 700-800 加粗

渐变规范

头部渐变: 珊瑚 → 浅珊瑚 → 暖杏 → 背景色
按钮渐变: 135deg, #FDDCC8 → #F2A99A → #E8836B
卡片背景: 纯白 #FFFFFF
页面背景: #FAF3ED 暖灰奶油

设计原则

1. 避免纯黑/纯白对比，使用暖色系灰阶
2. 减少直角，多用圆角营造亲和感
3. 使用暖色系投影而非冷灰投影
4. 渐变过渡要柔和，不要过于跳跃
5. 留白适度，不拥挤，有呼吸感
6. 价格使用主色珊瑚，不用刺眼红色
